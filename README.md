# Urban Brew Café Website ☕

This project is a modern website developed for a local café named "Urban Brew Café" located in Dewas.

## 🌐 Project Overview
The goal of this project is to help local businesses establish a strong online presence and attract more customers through a visually appealing and functional website.

## ✨ Features
- Modern and responsive UI design
- Interactive menu with item variants (Small / Medium / Large)
- Product images for better user experience
- Contact form
- Google Maps integration for location
- WhatsApp chat support for easy communication

## 📢 Project Pitch
I created this website for a local café to solve the problem of limited digital presence in small cities like Dewas. Many local businesses do not have websites, which reduces their visibility.

This website helps the café attract more customers, showcase its menu, and provide easy communication through WhatsApp and location access.

- HTML

- JavaScript
